Notion 연동 설정

Notion API 키와 데이터베이스 ID를 입력하면 노션 DB를 자동으로 불러옵니다.
설정값은 이 브라우저에만 저장됩니다.

  1. notion.so/my-integrations 에서 새 통합 만들기
  2. 생성된 Secret key 복사 → 아래 붙여넣기
  3. 노션 DB 페이지 → 오른쪽 위 ··· → Connections → 방금 만든 통합 연결
  4. DB URL에서 ID 복사: notion.site/여기32자리?v=...
* CORS 문제로 브라우저에서 직접 Notion API 호출이 막힐 수 있어요.
* 그럴 경우 아래 "샘플 데이터 보기"로 UI를 먼저 확인하세요.
* 실제 운영은 Vercel/Netlify 무료 배포 + serverless function 방식을 추천드려요. (도움 요청하시면 코드 만들어드려요!)
